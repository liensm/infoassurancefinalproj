# SecureText | A Cryptography Browser Extension
A browser extension that encrypts and decrypts text using:
- Caesar cipher
- Rail Fence cipher
- Caesar Backyard Fence cipher

It supports selecting and inputting highlighted text:
- Load highlighted text from a tab.
- Choose between Caesar, Rail Fence, Caesar Backyard Fence for encryption/decryption
- Copy transformed output to clipboard.

## Files
- manifest.json
- extension.html
- extstyling.css
- textretriever.js
- logic.js
