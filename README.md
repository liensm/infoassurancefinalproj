# Cipher Highlighter Extension

A browser extension that encrypts and decrypts text using:
- Caesar cipher
- Rail Fence cipher
- Caesar Backyard Fence cipher

It supports selecting and inputting highlighted text:
- Load highlighted text from a tab.
- choose between caesar, rail fence, combo
- Copy transformed output to clipboard.

## Files
- manifest.json
- extension.html
- extstyling.css
- textretriever.js
- logic.js

## Manual
1. Open Extensions page.
2. Enable Developer mode.
3. Choose Load unpacked.
4. Select this folder.
